# Angular

__Framework d'application web__ open-source basé sur `TypeScript`, dirigé par l'équipe `Angular` de __Google__ et maintenu par une communauté d'individus et d'entreprises.

---

### Caractéristiques

- __Architecture composantielle__
- __Système de routage__
- Client HTTP
- Gestion formulaires
- Injection de dépendances
- Utilitaires test

__Architecture composantielle__ : Construction de __composant réutilisable et indépendants__.

__Système de routage__ : Mécanisme navigation entre différentes vues de l'application __sans rechargement de page complète(SPA-Single Page Appllication)__.
```bash
URL                              → Composant affiché
─────────────────────────────────────────────────────
/                                → DashboardComponent
/consultants                     → ConsultantListComponent
/consultants/123                 → ConsultantDetailComponent
/consultants/new                 → ConsultantFormComponent
/missions                        → MissionListComponent
/missions/456                    → MissionDetailComponent
/login                           → LoginComponent
```
